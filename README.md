# 📊 JC 投資組合前瞻性壓力測試與風險監控系統 (JC Portfolio Dashboard)

這是一個專為台灣美股/台股雙軌槓桿投資者設計的**個人財務與資產壓力測試看板**。系統結合了即時股價同步、多情境債務模擬（LOC 隨借隨還 / 股票質押 / 信用貸款）、歷史資產自動對齊，以及公開資訊觀測站 (MOPS) 即時重大訊息預警功能。

---

## 🚀 核心功能特色

1. **📊 投資組合資產與風險看板**
   - **實時市值估算**：串接 Yahoo Finance API 動態同步持股最新股價，即時計算股票總市值。
   - **槓桿與波動度監控**：計算加權 Beta 值，模擬當大盤出現波段修正時，真實本金（ROE）與總資產的面臨波動。
   - **政策與利息支出門檻**：設定台灣央行重貼現率與台銀一年期定儲利率，動態推算負債的每年利息支出與生息門檻。

2. **🛡️ 槓桿壓力測試與危機模式 (Crisis-Mode Beta)**
   - 壓力測試模擬包含大盤跌幅、危機模式 Beta 下，資產淨值的即時變動。
   - **LOC 股票質押維持率監控**：動態追蹤維持率，安全線（🟢 166% 以上）、預警線（🟡 130%~166%）、追繳線（🔴 130% 以下，2 日內補繳）、以及斷頭清算線（💀 110%）。

3. **📈 每週資產歷史趨勢折線圖**
   - **自動對齊週日**：系統會自動將歷史記錄點對齊為「當週的週日」（週日最能反映週末收盤後的完整資產狀態）。
   - **數據快速補登與修改**：提供手動補登編輯器，防範雲端平台因重啟造成 ephemeral 磁碟資料遺失。

4. **📡 MOPS 即時重訊預警衛星**
   - 掃描您投資組合中所有個股近 30 天內在「公開資訊觀測站」發布的重大訊息與股利/除權息公告。
   - **HTTP 輕量級極速引擎**：移除繁重的 Playwright 無頭瀏覽器，100% 採用純 HTTP requests，免除 Streamlit Cloud 伺服器因超載 (OOM) 崩潰的風險。
   - **最新優先排序**：所有即時重訊皆依時間「由新到舊」排序，今日最新重訊以火苗符號（🔥）醒目置頂。

## 💾 數據持久化說明 (Streamlit Cloud ephemerality)

> [!IMPORTANT]
> **Streamlit Cloud 免費版容器的磁碟為「暫時性（Ephemeral）」**。
> 當 App 進入休眠重啟或部署新版本時，容器會重建，您在網頁上新增/修改的資產 CSV 數據將會遺失。

### 如何永久保存您的資產數據：
1. 當您在網頁上利用 `手動編輯器` 修改了持股或歷史資產後，點選 Sidebar 下方的 **「📥 下載最新 CSV 檔案」**。
2. 將下載回來的 `portfolio_data.csv`、`loans_data.csv` 或 `asset_history.csv` 覆蓋您本機端的檔案。
3. 將這份變更 commit 並 **push 回您的 GitHub 倉庫**。
4. Streamlit Cloud 會自動讀取您 GitHub 上最新的檔案，數據就永遠不會遺失了！

## 📂 專案檔案結構

- `app.py`: 看板的主程式（Streamlit 應用程式）。
- `portfolio_data.csv`: 您的持股數據，包含代號、均價、股數、已實現損益。
- `loans_data.csv`: 您的借貸數據，包含 principal、利率、借款類別。
- `asset_history.csv`: 您的每週歷史資產淨值記錄（對齊週日）。
- `stocks_list.txt`: **自癒型股票代號中文對照表**。若此檔案遺失或為空，系統在啟動時會**自動向證交所與櫃買中心 OpenAPI 請求**並自動建立該檔案。
- `requirements.txt`: Python 依賴套件清單。

## 🔧 本地運行指南

若您想在自己的電腦上運行此看板：

1. **複製專案**：
   ```bash
   git clone <your-repo-url>
   cd portfolio-dashboard
   ```

2. **安裝依賴套件**：
   ```bash
   pip install -r requirements.txt
   ```

3. **啟動 App**：
   ```bash
   streamlit run app.py
   ```
   瀏覽器會自動開啟 `http://localhost:8501`。
