# 📊 JC 投資組合前瞻性壓力測試與風險監控系統

這是一個專為台股與美股投資人設計的**個人資產與風險管理看板**。特別適合有使用槓桿工具（如股票質押、信用貸款、證券借貸）的投資者。它能幫助您動態監控資產健康度、模擬極端市場崩盤下的財務狀況，並即時追蹤持股的重大訊息與利息支出。

---

## 🌟 這款工具能幫您做什麼？ (核心功能)

### 1. 📈 投資組合資產與負債即時監控
* 系統會自動拉取最新美股與台股的即時價格，計算您的持股總市值。
* 整合您登記的各項借貸（如質押、信貸），一目了然呈現您的**總資產、總負債、槓桿成數與資產淨值**。

### 2. ⚡ 槓桿維持率與「斷頭」安全預警
* **即時監控股票質押維持率**：
  以清晰的燈號提示目前的維持率狀態，防範因市場波動導致被追繳或斷頭清算：
  * 🟢 **安全線 (166% 以上)**：資產狀況健康。
  * 🟡 **預警線 (130% ~ 166%)**：應密切注意大盤震盪。
  * 🔴 **追繳線 (130% 以下)**：面臨被要求補繳保證金的風險。
  * 💀 **清算線 (110% 以下)**：達面臨斷頭處分門檻。

### 3. 💥 模擬極端市場崩盤的「壓力測試」
* 您可以手動拉動滑桿，模擬大盤下跌 10%、20% 甚至 50% 時的極端情況。
* 系統會立即計算並顯示：**資產會縮水多少、維持率會降到多少、是否會觸發追繳**。讓您在危機發生前，提早做好資金調度防備。

### 4. 📉 每週資產淨值歷史趨勢
* 提供視覺化折線圖，記錄並呈現您的資產每週的成長軌跡。
* 內建手動補登編輯器，方便您隨時修改或補登過去的資產數據。

### 5. 🔥 持股重大訊息即時通知 (台股)
* 系統會自動掃描您的台股持股，抓取「公開資訊觀測站」最近 30 天內發布的重大訊息。
* 今日最新的重訊會以火苗（🔥）醒目置頂，方便您第一時間掌握公司除權息、財務報告等關鍵消息。

## 💾 數據保存說明 (如何讓資料永久存檔？)

本系統預設執行於雲端免費容器上，當看板重啟或休眠時，您在網頁上編輯的資產資料可能會遺失。若要永久保存您的個人資料：

1. 在網頁上新增/修改資產或借貸資料後，點選側邊欄下方的 **「📥 下載最新 CSV 檔案」**。
2. 將下載的檔案覆蓋您本機端專案中的 `portfolio_data.csv`、`loans_data.csv` 或 `asset_history.csv`。
3. 將這些檔案 commit 並 **push 回您的 GitHub 倉庫**。網頁看板會自動讀取 GitHub 上的最新檔案，資料便永遠不會遺失。

---

## 📂 專案檔案說明

* `app.py`：看板的主程式。
* `portfolio_data.csv`：您的持股資料庫（包含代號、均價、股數等）。
* `loans_data.csv`：您的借貸資料庫（包含金額、利率、借款類別等）。
* `asset_history.csv`：每週歷史資產淨值記錄。
* `stocks_list.txt`：股票代號與中文對照表（系統會自動生成並更新）。
* `requirements.txt`：專案依賴套件清單。

## 🛠️ 快速安裝與啟動

### 1. 安裝套件
請打開終端機 (Terminal)，切換到本專案資料夾，執行：
```bash
pip install -r requirements.txt
```

### 2. 啟動看板
執行以下指令，系統會自動在瀏覽器打開看板介面：
```bash
streamlit run app.py
```
啟動後會自動開啟 `http://localhost:8501`。
